Growth

This project was the same assignment but I created two projects instead. These projects both show of my ability to creat images using turtle programming and how to have different functions applied to have different desgins or images.

This work represents my area of growth by showing I can create a scene of a city by only using turtle programming. Drawing all of the buildings and clouds was very difficult because every time I would generate a new scene the clouds would be vertically instead of being horizontal. Also since I have three types of buildings I did not want some to float too high and I did not want some to be covered which was difficult because I had to redo my code multiple times. Creating the windows and keeping them contained in a certain area was very difficult because I wanted to have the windows on the buildings but they still floated too high, so I lowered the x and y to have them on the buildings. My effort on this project helped the success was very important because if I had given up and decided to create a different design, it would not have turned out to be as good as it is. I would like to change the windows to be applied to each row of buildings instead of them just being spread out at random.

This work represents the area of growth by showing I can create a tower and lanterns. In this specific scene I had to use a turtle to draw my tower. This portion was the most challenging because I used a ton of code to move the turtle and it was very repetitive. The tower at first would look great but once I ran the code by adding the lanterns the whole tower would become deformed. The lanterns were also a very challenging part of this project because they would become messed up once I ran my project and would become a circle or the purple lines would spread out all across my screen. I had to change my code and directions multiple times and use many different amounts of length to move by. At first I did abandon this idea and went to create my sunset project but after a while I came back to finish my original idea and even if it took a lot of effort and energy it ended up as a very appealing scene. I would like to add more details to the tower because in the movie Tangled it this a very beautiful and wonderful tower while merrily the structure.
